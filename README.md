# Vince's App

A simple Android application for managing friends and tracking transactions.

## Features

*   **Dashboard**: Get a quick overview of your recent activity.
*   **People**: Manage a list of your friends and view their details.
*   **History**: See a complete history of all your transactions.
*   **Settings**: Configure the app to your liking.

## Tech Stack

*   **100% Kotlin**: The entire app is written in modern Kotlin.
*   **Jetpack Compose**: The UI is built with Jetpack Compose, Android's modern toolkit for building native UI.
*   **MVVM Architecture**: The app follows the Model-View-ViewModel architectural pattern.
*   **Room**: For local data persistence.

## How to Build

1.  Clone the repository.
2.  Open the project in Android Studio.
3.  Build and run the app.
